2. Descrizione del servizio

RiskFlow è un'applicazione web per trader che fornisce:

  • Strumenti di position sizing basati sul capitale e sul rischio percentuale
  • Visualizzazione grafica di Entry, Stop Loss e Take Profit su chart
  • Integrazione con exchange di criptovalute (Bitget, Bybit, Weex, BingX) tramite API
  • Monitoraggio del saldo e delle posizioni aperte
  • Strumenti di gestione del rischio (SL Shield, risk/reward calculator)

Il Servizio è attualmente in fase Beta. Funzionalità, interfaccia e disponibilità possono variare senza preavviso.

4. API Keys e integrazione exchange

RiskFlow ti permette di collegare le API Keys dei tuoi exchange per leggere saldo e posizioni, e per inviare ordini. A tal proposito:

  • Le API Keys sono memorizzate localmente nel tuo browser (localStorage), accessibili solo a te sul tuo dispositivo
  • Sei l'unico responsabile della configurazione delle API Keys (permessi, restrizioni IP)
  • Raccomandiamo di configurare le chiavi con soli permessi di trading, disabilitando prelievi e trasferimenti
  • RiskFlow non è responsabile per perdite derivanti da chiavi con permessi eccessivi
  • Puoi eliminare le API Keys in qualsiasi momento dall'app
Best practice sicurezza: Crea API Keys dedicate per RiskFlow, con restrizione IP al dominio del servizio, e senza permessi di prelievo.
