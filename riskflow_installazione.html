Passi di installazione
1
Apri la cartella dati di MetaTrader 5
Avvia MetaTrader 5. Dal menu in alto clicca su File → Apri cartella dati. Si aprirà una finestra di Esplora File nella directory di installazione di MT5.
File → Apri cartella dati
2
Naviga fino alla cartella degli Expert Advisors
Nella finestra di Esplora File appena aperta, entra nella cartella:
MQL5 / Experts /
Puoi anche creare una sottocartella dedicata, ad esempio Experts/RiskFlow/, per tenere l'EA separato dagli altri.
3
Copia il file RiskFlow_EA.ex5 nella cartella
Incolla il file RiskFlow_EA.ex5 nella cartella MQL5/Experts/ (o nella sottocartella che hai scelto). I file .mqh non sono necessari per far girare l'EA.
⚠
Assicurati di copiare il file .ex5 e non il file .mq5 (sorgente). Il file già compilato è quello con estensione .ex5.
4
Ricarica gli Expert Advisors in MT5
Torna in MetaTrader 5. Nel pannello Navigator (aprilo con Ctrl+N se non è visibile), fai clic destro su Expert Advisors e scegli Aggiorna. L'EA apparirà nell'elenco.
Ctrl + N apre il Navigator
5
Trascina l'EA su un grafico aperto
Nel Navigator, espandi Expert Advisors e trova RiskFlow_EA. Trascinalo direttamente sul grafico del simbolo che vuoi usare (es. EURUSD, BTCUSD…). Si aprirà la finestra di configurazione.
6
Configura i parametri iniziali
Nella finestra che si apre, vai sulla scheda Ingressi e imposta i valori in base alla tua operatività:
ParametroDefaultDescrizione
InpRiskPct 0.2 Rischio per trade in % sul bilancio. Es. 0.5 = 0.5%
InpDLLPct 2.0 Limite di perdita giornaliera % (Daily Loss Limit)
InpDLLActive false Attiva il blocco automatico al raggiungimento del DLL
InpSLPActive false Attiva lo SL Shield (protezione stop loss)
InpSpreadInSL true Include lo spread nel calcolo della distanza SL
InpFeeTaker 0.0 Commissione taker % (per crypto/CFD con fee)
InpConfirmTrade true Mostra finestra di conferma prima di aprire ogni ordine
InpPanelScale 130 Scala grafica del pannello (valori: 100–170)
InpTextScale 100 Scala del testo nel pannello (valori: 50–130)
InpMagic 202401 Magic number per identificare gli ordini dell'EA
InpSlippage 3 Slippage massimo consentito in punti
Consiglio: se usi più EA contemporaneamente, cambia InpMagic con un numero diverso per ognuno, così non si confondono tra loro.
7
Abilita il trading automatico in MT5
Vai sulla scheda Common nella stessa finestra di configurazione e assicurati che la voce "Consenti trading automatico" sia spuntata. Poi clicca OK.
⚠
Controlla anche il pulsante AutoTrading nella barra degli strumenti di MT5 (deve essere verde e attivo), altrimenti l'EA non potrà aprire ordini.
✓
Verifica che il pannello sia attivo
Se tutto è andato a buon fine, il pannello grafico di RiskFlow apparirà sul lato destro del grafico, con il tema navy/viola. In basso a destra del grafico vedrai anche la faccina sorridente (😊) che indica che l'EA è in esecuzione.
✓
Il messaggio "RiskFlow v2 MT5 pronto" comparirà nella barra delle notifiche in basso — installazione completata!
🚀
Pronto per fare trading RiskFlow EA è installato e attivo. Consulta la guida all'utilizzo per imparare a impostare SL, TP e aprire ordini con il pannello.
